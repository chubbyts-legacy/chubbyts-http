## Description

Http Request/Response with middleware and handler, [PSR-15][2] inspired.

## Requirements

 * node: 16
 * [chubbyts/chubbyts-http-types][3]: ^1.2.3
 * [qs][4]: ^6.11.1

## Installation

Through [NPM](https://www.npmjs.com) as [@chubbyts/chubbyts-http][1].

```ts
npm i @chubbyts/chubbyts-http@^1.2.1
```
